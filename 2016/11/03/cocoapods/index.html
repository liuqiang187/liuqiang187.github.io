cocoapods升级到1.2.0beta

2016-11-03

tools

首先升级 gem

gem -v 查看版本，最好2.6.0以上
gem update --system 更新版本

如果提示错误You don’t have write permissions for the /Library/Ruby/Gems/2.0.0 directory.
请用sudo gem update --system

等待更新完成

替换 gem sources

gem sources -l 先查看当前source
gem sources --add https://gems.ruby-china.org/ --remove https://rubygems.org/

先 add ruby-china.org 再 remove 当前source
再次gem sources -l确保只有 gems.ruby-china.org

更新cocoapods

sudo gem install -n /usr/local/bin cocoapods --pre 执行更新
pod --version 待更新完后查看版本

pod setup 设置仓库
这个过程比较漫长

成功！！可以使用Swift3.0的三方库了
